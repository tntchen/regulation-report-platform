## 6. 已知边界（不是 bug，测试时不必惊讶）

1. **Mock 模式生成的是 G01 形态 SQL**：新建 G11 任务直连跑会在质量门禁被拦（Mock 无 G11 分支）——这是种子脚本预造 TASK_DEMO_G11_001 的原因；走查 3.4 时若挂起任务续跑失败属预期，演示主用预置任务
2. 制度 diff 只支持 TXT/MD 文档
3. waiting_confirmation 状态下取消任务不会即时终结（遗留项）
4. 接口文件行数受只读通道行数上限截断（演示数据量小，不会触发）
5. 台账"已逾期"是查询时懒计算——把月份切到过去月份即可看到红色逾期样例
